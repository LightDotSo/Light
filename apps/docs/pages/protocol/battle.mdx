# Team Battles

Through this playground, Team Battles captures value for $LIGHT token in 2 ways:

- Direct: Increasing buy Demand for $LIGHT; each participant must create a personal profile, spending 1 $LIGHT.
- Indirect: Strongly incentive for promoting activity on Light, generating more attention and [Super Activities](/protocol/battle), thereby indirectly capturing value into $LIGHT tokens.
